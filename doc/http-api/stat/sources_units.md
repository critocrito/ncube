# Count of Sources of Units

Show the count of different sources of units for a workspace.

**URL** : `/workspaces/<workspace_id>/stats/data/sources`

**Method** : `GET`

## Success Response

The response envelope contains the stat value.

**Code** : `200 OK`

**Content examples**

```json
{
  "status": "success",
  "data": 5
}
```
