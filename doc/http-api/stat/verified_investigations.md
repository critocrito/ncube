# Verified Count Investigations

Show the count of verified data for an investigation.

**URL** : `/workspaces/<workspace_id>/stats/investigations/<investigation>/verified`

**Method** : `GET`

## Success Response

The response envelope contains the stat value.

**Code** : `200 OK`

**Content examples**

```json
{
  "status": "success",
  "data": 23
}
```
