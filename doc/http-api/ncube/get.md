# Show Ncube Configuration

Display the configuration of the Ncube host application.

**URL** : `/`

**Method** : `GET`

## Success Response

**Code** : `200 OK`

**Content examples**

```json
{
    "status": "success",
    "data": {
        "workspace_root": "~/ncube",
        "name": "Christo",
        "email": "christo@cryptodrunks.net"
    }
}
```
