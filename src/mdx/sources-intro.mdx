Sources are links to data that you would like to preserve. Examples for a source are a Youtube channel, a Tweet or a website URL. You can enter links manually or use the [Discovery web extension](https://sugarcubetools.net/discovery/installation) to send links of interest to Ncube with the click of one button without having to leave the browser environment.

Follow this link to read more about [sources](https://sugarcubetools.net/ncube/glossary#source).
